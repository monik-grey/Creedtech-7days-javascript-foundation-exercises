**🔁 FOR LOOPS**
1️⃣ Reverse Without Cheating
Create an array of 10 numbers.
Using a for loop, print the numbers in reverse order.
⚠️ You are NOT allowed to use .reverse().

2️⃣ Even or Odd?
Create an array with at least 15 numbers.
Using a for loop:
Count how many numbers are even.
Count how many numbers are odd.
Print both results clearly.

3️⃣ Pattern Builder (Slightly Tough)
Using nested loops, print this pattern in the console:

- \*\*
  ***
  ***
  ***

No shortcuts. Use loops properly.

**🔄 WHILE LOOPS**
1️⃣ Countdown With a Twist
Using a while loop:
Start counting down from 20 to 1.
Skip any number that is divisible by 3.
Print the rest.

2️⃣ Find the Secret Number
Create a variable called secretNumber and set it to 7.
Now:
Start another variable at 1.
Keep increasing it using a while loop.
Stop when it matches the secret number.
Print how many attempts it took.

**🧠 FUNCTIONS**
1️⃣ Analyze Scores
Create a function called analyzeScores.
It should:
Accept an array of numbers.
Return the highest score.
Return the lowest score.
Return the average score.
⚠️ Do NOT use Math.max() or Math.min().

2️⃣ Password Checker
Create a function called validatePassword.
Rules:
Password must be at least 8 characters.
Must contain at least one number.
Return "Valid" or "Invalid".
Make it smart. Don’t hardcode anything.

3️⃣ Multiply Everything
Create a function that can accept multiple numbers and return their total multiplication.
It should work whether you pass 2 numbers or 10 numbers.

**📦 ARRAYS**
1️⃣ Filter Manually
Create an array of numbers.
Without using .filter():
Create a new array that contains only numbers greater than 10.

2️⃣ Remove Duplicates (Manually)
Create an array that contains duplicate numbers.
Without using Set:
Remove the duplicates.
Return a clean array.

**🧱 OBJECTS**
1️⃣ Student Record
Create an object called student with:
name
age
courses (array)
isGraduated (boolean)
Now:
Use a loop to print all the keys and values.

2️⃣ Store Inventory
Create an object that represents a product:
name
price
quantity
Write a function that calculates the total value of the product (price × quantity).

**🧩 DESTRUCTURING**
1️⃣ Array Destructuring
Given an array of 4 items:
Extract the first two into separate variables.
Store the remaining items in another variable.

2️⃣ Object Destructuring
Given an object with 3 properties:
Extract two properties.
Rename one of them while destructuring.

**🚀 SPREAD OPERATOR**
1️⃣ Merge and Extend
Create two arrays.
Use spread to:
Merge them.
Add two extra items at the end.

2️⃣ Clone and Update
Create an object.
Use spread to:
Clone it.
Change just one property in the new version.

**🌳 REST OPERATOR**
1️⃣ Sum Function
Create a function that accepts any number of arguments and returns their total sum.

2️⃣ First vs The Rest
Create a function that:
Takes at least one argument.
Separates the first argument from the rest.
Prints both separately.

**📂 Assignment Submission Guide (Important)**

Please follow the steps carefully:

**2️⃣ Clone Your Fork**

**3️⃣ Organize Your Work**

Inside the repository:

- while-loops

- functions

- arrays

- objects

- destructuring

- spread

- rest

- dom-project (This folder has sub folders - Fancy Counter & Word Analytics)

Write your JavaScript logic inside the appropriate folder.

Keep your code clean and properly structured.

**4️⃣ Commit and Push**

Track your work properly:

Add your changes

Write meaningful commit messages

Push to your forked repository

**5️⃣ Open a Pull Request (PR)**
